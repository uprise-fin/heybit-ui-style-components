# 레이아웃을 구성법

### 레이아웃을 구성할때는, Section, Container, Grid 로 구성됩니다.

- Section : width 전체 영역을 할당함으로써, 배경색을 주어질때 사용하게 됩니다.
- Container : content-width를 지정합니다. (기본, md, sm) 사이즈가 있습니다.
- Grid : 2단으로 화면을 구성할때 사용합니다.

### 레이아웃을 구성한 후 Container에 해당하는 아이템들을 사용해서 화면을 구성 합니다.

- Card : 카드 형태
- Panel : 일반 흰색 바탕 화면

### 기본 레이아웃을 잡는 방법.

<div class="section">
  <div class="container p">
    <div class="panel">
      <h1>Title 입니다.</h1>
      내용 내용 내용 내용 내용 내용 내용 내용 내용
    </div>
  </div>
</div>

### 섹션 전체에 배경을 사용하는 방법

<div class="section p">
  <div class="container">
    <div class="card success">
      <div class="card-heading">Title</div>
      <div class="card-body">내용 내용 내용 내용 내용 내용 내용 </div>
      <div class="card-footer">footer footer footer </div>
    </div>

  </div>
</div>

### 2단으로 사용하는 방법

<div class="section">
  <div class="container">
    <div class="row">
      <div class="col">
        <h1>Title 입니다.</h1>
        내용
      </div>
      <div class="col">
        <h1>Title 입니다.</h1>
        내용
      </div>
    </div>
  </div>
</div>
